## 🎯 Guia Rápido

### Setup Frontend
```bash
# 1. Navegar para frontend
cd ../LEP-Front

# 2. Instalar dependências
npm install

# 3. Revisar FRONTEND_INSTRUCTIONS.md
cat docs/frontend/FRONTEND_INSTRUCTIONS.md

# 4. Iniciar desenvolvimento
npm run dev
```

### Desenvolver Cardápio
```bash
# Ver instruções específicas
cat docs/frontend/FRONTEND_CARDAPIO_PROMPT.md
```
